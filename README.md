# Oddworld: New 'n' Tasty Archipelago

An implementation of Oddworld: New 'n' Tasty into the [Archipelago Multiworld Randomiser](https://archipelago.gg/), based upon the example APQuest world.

The source of the actual world can be found on my fork of the Archieplago project [here](https://github.com/Knuxfan24/Archipelago).
